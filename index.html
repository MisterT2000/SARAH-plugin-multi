Le plugin Jarvis est un pluguin destiner a controler, et naviguer sur votre PC, il permet d'ouvrir des dossiers, documents, lancer des logiciels (jeux, editeur de texte...),
ouvrir des pages web, agir sur les périphériques du PC.
Pour que Sarah puisse ouvrir les logiciels il est important de modifier les adresses de ces derniers dans les fichiers .bat du dossier bin
se trouvant lui même dans le dossier du plugin : Plugins/jarvis/bin.

Fonctionnement du pluguin :

Le xml se compose de commande de ce genre :

<item>[commande]<tag>out.action.val="[valeur]"</tag></item> commande : suite de mors reconnus par le client.
valeur : valeur lu dans par le fichiers js qui renseigne au fichier js ce qu'on lui demande.

Voiçi un exemple <item>affiche ton interface<tag>out.action.val="runjarvis"</tag></item> Dans le fichiers js les commandes son lus de cette façon : case "[valeur]": var process = '%CD%/plugins/jarvis/bin/[fichier cible]'; var Txt = new Array; Txt[0] = "[texte de la réponse]"; break; valeur : on retrouve donc la valeur que l'on a renseigné dans la commande du xml si on reprend notre exemple la valeur serait runjarvis.
fichier cible : nom du fuchier .bat .vbs ou ce que vous voudrez qui sera executé.
texte de la réponse : liste de phrase que sarah dira de façon aléatoire de façon a casser la monotonie des réponses
et de la rendre un peu plus humaine.


Voici ce que donnerais le code js qui correspond à l'exemple plus haut pour le xml case "runjarvis": var process = '%CD%/plugins/jarvis/bin/runjarvisbat'; var Txt = new Array; Txt[0] = "j'ai affiché mon interface"; Txt[1] = "je m'en auccupe"; Txt[2] = "tout de suite"; Txt[3] = "c'est fait"; break;

Comme vous le voyez vous pouvez renseigné plusieurs réponse pour la réponse aléatoire.
il est important de respecter la structure var Txt = new Array; Txt[0] = "phrase 1"; Txt[1] = "phrase 2"; Txt[2] = "phrase 3"; Txt[3] = "phrase 4";

fonctions possible :

Sarah, tu es là ? :

  • elle répond "une des réponses aléatoires, cette commande sert à vérifier que sarah vous entend, qu'elle est en route ou en veille.
gestions des périphériques
  • Sarah, éteinds l'écran : Sarah éteint l'écran
  • Sarah, ouvre le DVD : Sarah ouvre le lecteur DVD de votre PC
  • Sarah, change la sortie audio : Sarah passe le son du PC des enceinte à votre casque ou ampli (nécessite le logiciel Soundswich
